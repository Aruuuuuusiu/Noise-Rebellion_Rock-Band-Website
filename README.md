# NOISE REBELLION

A visually immersive **alt-rock band website** built using **HTML, CSS, and vanilla JavaScript**, designed to showcase modern front-end design, creative animations, and interactive UI effects.

---

## Project Overview

**NOISE REBELLION** is a fully responsive, visually immersive alt-rock band website built with HTML, CSS, and vanilla JavaScript. It features a bold, glitch-inspired design with smooth scrolling, animated sections, interactive galleries, simulated music playback, tour listings with modal interactions, merch previews with quick-view modals, and dynamic UI effects.

The project focuses on **modern front-end design**, **creative animations**, and **user interaction**, making it a strong showcase of aesthetic web design, DOM manipulation, and responsive layouts — all **without using external JavaScript frameworks**.

> **Note:**  
> This is a personal portfolio project created for learning and demonstration purposes.

---

## Features
> **Note:**  
> Some features are currently being made.


-  Glitch-inspired typography and CSS animations  
- Fully responsive design (desktop, tablet, mobile)
- Fixed navigation with scroll-based styling
- Smooth scrolling navigation with offset handling
- Simulated music playback UI (non-audio)
- Filterable image gallery with animated transitions
- Video showcase cards with play overlays
- Tour dates with interactive ticket modals
- Merch preview with “Quick View” modal popups
- Custom notification system (success / info feedback)
- Keyboard shortcuts for section navigation
- Hidden Easter egg (Konami code)

---

## Tech Stack

- **HTML5** — Semantic structure  
- **CSS3** — Custom animations, responsive layouts, modern styling  
- **JavaScript (Vanilla)** — DOM manipulation, modals, notifications, keyboard shortcuts, UI interactions

No external frameworks or libraries are used (except icon fonts and web fonts).
